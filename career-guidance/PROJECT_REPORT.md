## ❓ TROUBLESHOOTING

### Server Won't Start:
```bash
npm run dev
```

### Changes Not Showing:
- Refresh browser (Ctrl+R)
- Clear browser cache
- Restart server

### Port 3000 Already in Use:
- Kill process on port 3000
- Or use different port: `npm run dev -- --port 3001`

### Module Errors:
```bash
rm -rf node_modules
npm install
npm run dev
```
